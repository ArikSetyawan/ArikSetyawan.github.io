My Portfolio



  • Work-log & Cycletime Application (2023)
    I worked on this project when I worked at PT Lestari Mulia Sentosa. This project is about logging employee process time and cycle time for production. By collecting this data, we can analyze each employee performance and process time for each product that processed, so the area working leader can know what todo to improve quality, quantity, process time, employee morale, and etc. The way this application works is by scanning QR-code of transaction that worked on. Then employee can press start button to start the timer. Then if the employee want to pause, they can just press the pause button an pick a reason for why they have to pause it. When employee done after working for the transcation, they can press stop button and if any defect in product they workedon, they can input it how much the defect and description of it.
    Technology : Dart, Flutter, Figma, BLOC State Management, Isar, RESTful-API, Mobile Scanner, Workmanager.
    Demo : Lestari Worklog & Lestari Worklog Galery
